-- 삼항 연산자
일단 형태는,
[ 조건문   ] ? A선택지 : B선택지      이렇게 생겼어요!
조건문의 결과가 true라면 A선택지를, false라면 B선택지를 선택하게 돼요.
예를 들어,

int num = 3;
String s = num == 4 ? "맞았다" : "틀렸다"; 라고 한다면, 
num은 현재 3이라는 값으로 정의되어있으므로 num == 4는 false가 되고, 이에 문자열 변수 s에는 "틀렸다"가 담기게 되는거에요!

또 다른 예시로,
Scanner sc = new Scanner(System.in);
int input = Integer.parseInt(sc.nextLine());

int result = input > 5 ? 10 : input;
이런 식으로 사용한다면! 5 초과의 값을 입력받으면 변수 result에는 항상 10이라는 값이, 5이하의 값을 입력받으면 입력 받은 값 그대로 result에 저장할 수가 있어요!

이것은 
if( input > 5 ) {
	result = 10;
}else {
	result = input;
}
이렇게 작성된 코드와 같아요. 코드를 단축시킬 수 있는 장점이 있어요 ㅎㅎ



-- Random 클래스
자바에는 Random 클래스가 있어요!
내장된 util패키지 밑에 존재하는 Random클래스는 난수를 생성할 수 있어요.
사실 컴퓨터는 무척이나 멍청하기 때문에, 사람처럼 직접적으로 난수를 떠올려서 만들 수는 없어요! 마치 난수가 발생한 것 처럼 보이게끔 만들어주는 잘 짠 알고리즘의 결과물일 뿐이지,
실제 세계에 존재하는 난수와는 차이가 있다는 것을 꼭 기억해야해요! 회사마다 난수를 생성하는 클래스를 따로 만들어서 사용해요 ㅎㅎ
그나마 자바에서 기본적으로 제공해주는 이 Random 클래스에 의한 난수는 C언어나 다른 언어들과는 달리 꽤 괜찮은 난수를 만들어낸다고 해요 ㅎㅅㅎ(중복이 최소화됨)

사용법은,
Random r = new Random();
int number = r.nextInt(10);
이렇게 작성해주면 변수 number에는 0부터 9까지의 값 중 하나의 값이 랜덤하게 저장돼요 ㅎㅅㅎ
